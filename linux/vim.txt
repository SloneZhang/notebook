Vim 是一个功能强大的文本编辑器，它分为三种模式：普通模式、插入模式和命令模式。以下是一些基本的 Vim 命令和快捷键：

### 普通模式（Normal Mode）
- `i`：进入插入模式，在光标前插入文本。
- `I`：在当前行首插入文本。
- `a`：在光标后插入文本。
- `A`：在当前行末尾插入文本。
- `o`：在当前行下方新开一行并进入插入模式。
- `O`：在当前行上方新开一行并进入插入模式。
- `x`：删除光标所在位置的字符。
- `dd`：删除当前行。
- `yy`：复制当前行。
- `p`：粘贴复制的文本到光标后。
- `P`：粘贴复制的文本到光标前。
- `u`：撤销上一次操作。
- `Ctrl + r`：重做上一次被撤销的操作。
- `:q`：退出 Vim。
- `:wq` 或 `:x`：保存文件并退出 Vim。
- `:q!`：不保存文件强制退出 Vim。
- `:w`：保存文件但不退出 Vim。
- `:e <filename>`：打开一个文件进行编辑。
- `:%s/old/new/g`：将文件中所有的“old”替换成“new”。
- `:n` 或 `:next`：在多个文件中跳到下一个文件。
- `:N` 或 `:prev`：在多个文件中跳到上一个文件。
- `:bd`：关闭当前的缓冲区（文件）。
- `:ls` 或 `:buffers`：列出所有打开的文件。
- `:b <number>`：跳转到指定编号的缓冲区。
- `:sp <filename>`：水平分割窗口并打开文件。
- `:vsp <filename>`：垂直分割窗口并打开文件。
- `Ctrl + w` + `w`：在分割的窗口之间移动光标。
- `Ctrl + w` + `q`：关闭当前分割窗口。
- `:help <command>`：查看指定命令的帮助文档。
- `:help`：打开帮助文档。

### 插入模式（Insert Mode）
- `Esc`：退出插入模式，回到普通模式。

### 命令模式（Command Mode）
- 在普通模式下输入 `:` 进入命令模式，可以输入各种命令。

### 移动光标
- `h`：向左移动光标。
- `j`：向下移动光标。
- `k`：向上移动光标。
- `l`：向右移动光标。
- `w`：移动到下一个单词的开头。
- `b`：移动到当前单词的开头。
- `e`：移动到当前单词的末尾。
- `0`：移动到当前行的开头。
- `$`：移动到当前行的末尾。
- `gg`：移动到文件的第一行。
- `G`：移动到文件的最后一行。
- `Ctrl + u`：向上移动半页。
- `Ctrl + d`：向下移动半页。
- `Ctrl + f`：向下移动一整页。
- `Ctrl + b`：向上移动一整页。
- `{`：移动到上一个段落的开头。
- `}`：移动到下一个段落的开头。

这些是 Vim 的一些基本命令，掌握这些可以大大提高你的编辑效率。Vim 的功能非常丰富，还有许多高级命令和技巧等待你去探索。
